# Задача StructMining: Добыча структур (60 баллов)

Восстановите тип структуры S, исходя из таблицы выражений, приведенной ниже. Следует считать, что все выражения корректны, не вызывают ошибок компиляции, предупреждений и неопределенного или зависящего от реализации поведения. Между полями структуры S отсутствуют выравнивающие байты. Для представления знаковых целых используется дополнительный код.

В качестве ответа ожидается объявление структуры S.

```c
┌───────────┬────────┐
│ Тип       │ sizeof │
├───────────┼────────┤
│ short     │ 2      │
│ int       │ 4      │
│ long      │ 8      │
│ long long │ 8      │
│ void *    │ 8      │
└───────────┴────────┘
```

```c
struct S {
    /* Lost fields */
} a = {0};

char *x;
int *z;
```

```c
┌────────────────────────────┬───────────┬──────────┐
│ Выражение                  │ Тип       │ Значение │
├────────────────────────────┼───────────┼──────────┤
│ sizeof(a = *a.x)           │ size_t    │ 48       │
│ a.nums[13] + a.nums[1]     │ int       │ 0        │
│ (char *)&a.x - (char*)&a.z │ ptrdiff_t │ 8        │
│ (a.n = -1) > UINT_MAX      │ int       │ 1        │
│ z = a.z                    │ int *     │ 0        │
│ (char *)&a.z - (char*)&a.n │ ptrdiff_t │ 32       │
│ sizeof(a.nums)             │ size_t    │ 24       │
│ x = a.z                    │ char *    │ 0        │
└────────────────────────────┴───────────┴──────────┘
```

### Пример правильно форматированного ответа:

```c
struct S {
    int a;
    int b;
    int c;
};
```
