# Задача Overwatch: Китайский браслет (60 баллов)

**Ограничение времени:** 1 с  
**Ограничение решаемого времени:** 5 с  
**Ограничение памяти:** 64М  

## Условие задачи

У Ани сломался браслет для часов. Она купила новый, но тот оказался слишком велик для её тоненькой ручки.

Браслет состоит из звеньев, каждое из которых имеет положительную длину `size`, которая может различаться для разных звеньев. Аня хочет вынуть несколько **подряд идущих** звеньев из браслета, начиная с самого большого и некрасивого, чтобы он лучше сидел на руке.

Браслет описывается **двусторонним кольцевым списком**, каждый элемент которого имеет следующую структуру:

```c
struct segment {
    unsigned size;
    struct segment* next;
    struct segment* prev;
};
```

Напишите функцию:

```c
struct segment* fix(struct segment* head, unsigned size);
```

Функция принимает на вход список, описывающий исходный браслет, и обхват руки Ани (`size`). Функция должна вернуть новый список, получающийся из исходного удалением наибольшего возможного числа подряд идущих (в сторону `next`) элементов, начиная с наименьшего наибольшего размера в исходном списке, так, чтобы суммарный размер оставшихся сегментов был не меньше значения параметра `size`.

**Гарантируется**, что:
- суммарный размер остающихся сегментов не меньше, чем `size`,
- максимальный элемент определяется единственным образом.

Не забудьте освободить память, выделенную под удалённые элементы (гарантируется, что элементы исходного списка выделены в динамической памяти).

Размер исходного списка не превышает 1000.

## Оформление решения
Требуется написать только функцию, а не всю программу. Файл-посылка должен содержать исходную функцию, подключённые необходимые заголовочные файлы, вспомогательные функции (если есть). Выводить на экран ничего не нужно, проверяться будет возвращаемое значение функции. Наличие и содержание функции fail в файле будет игнорироваться.

## Шаблон файла-посылки:
```c
#include <stdlib.h>

struct segment* fix(struct segment* head, unsigned size) {
    // реализация функции
}
```

*Подсказка:* можно написать всю программу (с функцией main()) целиком, отладить её, а перед отправкой решения удалить или закомментировать функцию main().

## Примеры
**Входные данные**
```text
1 2 3 4 5
8
```

**Результат работы**:
```text
2 3 4
```
